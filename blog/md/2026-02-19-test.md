1）新建一个 md 文件

在终端执行：

cd ~/my-site
touch blog/md/2026-02-20-second.md


文件名建议用：YYYY-MM-DD-英文slug.md（以后好管理）

2）把文章内容写进去

用 VS Code 打开这个文件：

open -a "Visual Studio Code" blog/md/2026-02-20-second.md


粘贴一个模板（你改标题/日期/简介/正文即可）：

---
title: 第二篇：我开始认真写博客了
date: 2026-02-20
desc: 记录我搭建个人网站后的第一周，以及接下来会怎么持续更新。

## 这一周我做了什么
- 搭好个人站点
- 加了博客系统
- 做了导航组件化

## 接下来计划
- 每周至少更新一篇
- 写项目复盘和学习笔记


保存（⌘S）。

3）自动生成博客列表（关键一步）

在终端执行：

cd ~/my-site
python3 tools/gen_posts.py


它会自动把所有 blog/md/*.md 扫一遍，然后更新 blog/posts.js，把新文章加进列表里（并按日期倒序排序）。

4）提交并发布
git add blog/md/2026-02-20-second.md blog/posts.js
git commit -m "add: second blog post"
git push

5）看效果

打开：

博客列表：https://sicci-zzf.github.io/blog/

点进去会到：/blog/post.html?p=2026-02-20-second

小提醒：slug 要对得上

你文件名是：

2026-02-20-second.md

那链接里的 p= 就会是：

p=2026-02-20-second

（脚本会自动用文件名当 slug）
